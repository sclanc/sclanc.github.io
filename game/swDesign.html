Model

Each game object will have the following properties:

  • board: a two dimentional Java Script array of a user specified size used to keep track of player positions.
  • boardCheck: a two dimentional Java Script array of a user specified size used to keep track of recursion in checkBoard().
  • player1: player one's score.
  • player2: player two's score.
  • p1resign: a bool indicating whether or not player one has chosen to resign.
  • p2resign: a bool indicating whether or not player two has chosen to resign.
  • turn: a bool indicating whose turn it is (0 for p1, 1 for p2).

Functions

  • initGame() receives message from index.html, calls the createBoard() function, and creates the HTML table for the game while also initializing the gameState
  • createBoard() creates canvas object that acts as the visible board.
  • turn() receives board position and updates the gamestate accordingly, changing the turn bool, and calling the following functions:
    • insertGamePeice() inserts game peice into the DOM that is either white or black, depending on whose turn it is.
    • checkBoard() recursivly checks each intersection of the board, to make sure that any pieces that are in place are not entrapped. awards points as neccicary and calls the remove peice function as needed
    • removePeice() receives coordinates and removes game peice accordingly.
  • resign() recieves player, changes the value of coresponding player resign to true
  • resetResign() resets a player's resign to false, if the other player did not resign
  • endGame() if both players have resigned final scores are calculated and the game resets.
