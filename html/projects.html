Self generated news quiz content
Courantclose

This is a personal project derived for my love of the radio show, Wait Wait Don't tell Me. The project is written primarily in Python and currently gathers headlines from several different news sites, tokenizes them, and turns them into quiz questions. Next steps for the project include:

  • building a program that can turn the answers to the questions into crossword puzzles
  • building an API for the questions and crosswords
  • building a web app for the news based games
  • adding a real VS. fake news game
