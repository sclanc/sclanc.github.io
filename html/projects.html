Self generating news quizzes
Courantclose

This is a personal project derived from my love of the radio show, Wait Wait Don't Tell Me. The project is written primarily in Python and currently gathers headlines from several different news sites, tokenizes them, and turns them into quiz questions. Next steps for the project include:

  • building a program that can turn the answers to the questions into crossword puzzles
  • building an API for the questions and crosswords
  • building a web app for the news based games
  • adding a real vs. fake news game

Helping the Police
SecurCityclose

Some friends and I banded together to compete in CourtHack 2016, a hackathon for inovation in state courts. We built an Angular web app that integrated with Google Maps and court data to expediate the issuance of warrants. We won a substantial cash prize, and the app was picked up by a development firm in San Fransisco.
